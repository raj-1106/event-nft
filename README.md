A minimal Next.js application that connects to a Solana wallet and fetches NFTs owned by the connected wallet on **Solana Devnet**, using **Solana Wallet Adapter** and **Metaplex JS SDK**.

This project focuses on **clarity, structure, and correctness** over complexity.

---

## ‚ú® Features

- Connect Solana wallets (Phantom, etc.)
- Fetch NFTs owned by the connected wallet
- Display NFT name, image, and mint address
- Runs entirely on **Solana Devnet**
- Clean and minimal UI

---

## üß∞ Tech Stack

- **Frontend**: Next.js (App Router, TypeScript)
- **Wallet**: `@solana/wallet-adapter`
- **Blockchain**: Solana Devnet
- **NFT SDK**: Metaplex JS
- **RPC**: Solana Devnet public RPC
